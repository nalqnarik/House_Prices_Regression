# House Prices Prediction using Machine Learning

## Նախագծի նկարագրություն

Այս նախագծի նպատակն է կանխատեսել բնակարանների վաճառքի գինը (SalePrice)՝ օգտագործելով Kaggle-ի House Prices: Advanced Regression Techniques տվյալների հավաքածուն։

Քանի որ թիրախային փոփոխականը (SalePrice) շարունակական թվային արժեք է, խնդիրը հանդիսանում է Regression խնդիր։

Նախագծի ընթացքում իրականացվել են տվյալների նախնական մշակում, վիզուալիզացիա, ծայրահեղ արժեքների հայտնաբերում և հեռացում, ինչպես նաև մի քանի մեքենայական ուսուցման մոդելների կառուցում և համեմատություն։

---

## Տվյալների ուսումնասիրություն

Տվյալների հավաքածուն պարունակում է բնակարանների տարբեր բնութագրեր, այդ թվում՝

* Տան մակերես
* Սենյակների քանակ
* Ավտոտնակի չափ
* Տան ընդհանուր որակ
* Տան կառուցման տարեթիվ
* Տանիքի և արտաքին հարդարման տեսակները
* Այլ կառուցվածքային բնութագրեր

Կատարվել են հետևյալ քայլերը.

* Տվյալների չափերի ուսումնասիրություն
* Սյունակների տիպերի վերլուծություն
* Վիճակագրական ամփոփ տվյալների հաշվարկ
* Կոռելացիոն մատրիցի կառուցում
* Բացակայող արժեքների հայտնաբերում և մշակում
* Կատեգորիկ փոփոխականների վերափոխում One-Hot Encoding մեթոդով
* Նոր հատկանիշների ստեղծում

---

## Տվյալների նախապատրաստում

Նախքան մոդելների ուսուցումը կատարվել են հետևյալ քայլերը.

### Missing Values

Բացակայող արժեքները լրացվել են՝ կախված տվյալ սյունակի բնույթից։

### Categorical Encoding

Տեքստային սյունակները վերափոխվել են թվային ձևաչափի One-Hot Encoding-ի միջոցով։

### Outlier Detection

Հայտնաբերված անոմալ դիտարկումները հեռացվել են ուսուցման տվյալներից։

### Feature Scaling

KNN մոդելների համար կիրառվել է StandardScaler նորմավորումը։

---

## Կիրառված մոդելներ

Նախագծում կիրառվել են մի քանի տարբեր մեքենայական ուսուցման մոդելներ, որոնց արդյունքները համեմատվել են MAE, RMSE չափանիշների միջոցով։

### 1. Linear Regression

Linear Regression-ը հանդիսանում է ռեգրեսիոն խնդիրների ամենապարզ և ամենատարածված մոդելներից մեկը։ Այն ենթադրում է, որ անկախ փոփոխականների և թիրախային փոփոխականի միջև գոյություն ունի գծային կապ։

Առավելություններ.

* Պարզ և արագ իրականացում
* Հեշտ մեկնաբանվող արդյունքներ
* Հարմար է որպես բազային (baseline) մոդել

Թերություններ.

* Չի կարողանում լավ նկարագրել ոչ գծային կախվածությունները
* Զգայուն է արտասովոր արժեքների նկատմամբ

---

### 2. Lasso Regression

Lasso Regression-ը Linear Regression-ի ընդլայնված տարբերակն է, որն օգտագործում է L1 Regularization։

Օպտիմալացման ֆունկցիային ավելացվում է տուգանային անդամ, որի շնորհիվ որոշ գործակիցներ կարող են դառնալ զրո։ Այս հատկությունը հնարավորություն է տալիս իրականացնել feature selection։

Առավելություններ.

* Նվազեցնում է overfitting-ը
* Իրականացնում է ավտոմատ feature selection
* Հարմար է մեծ քանակությամբ հատկանիշներ ունեցող տվյալների համար

Թերություններ.

* Կարող է չափազանց մեծ regularization-ի դեպքում կորցնել օգտակար տեղեկատվություն

---

### 3. KNN Regressor

K-Nearest Neighbors (KNN) ռեգրեսիոն մոդելը կանխատեսում է նոր դիտարկման արժեքը՝ հիմնվելով տվյալ կետին ամենամոտ k հարևանների արժեքների վրա։

Ռեգրեսիայի դեպքում կանխատեսումը ստացվում է հարևանների թիրախային արժեքների միջինացման միջոցով։

Առավելություններ.

* Պարզ և ինտուիտիվ մեթոդ է
* Կարող է նկարագրել ոչ գծային կախվածություններ
* Չի պահանջում բարդ ուսուցման գործընթաց

Թերություններ.

* Զգայուն է feature scaling-ի նկատմամբ
* Մեծ տվյալների դեպքում կարող է աշխատել դանդաղ
* Բոլոր հարևաններին տալիս է հավասար կարևորություն

Նախագծում KNN մոդելի համար կիրառվել է StandardScaler նորմավորում, որպեսզի բոլոր հատկանիշները լինեն նույն մասշտաբի վրա։

---

### 4. Weighted KNN Regressor

Weighted KNN-ը KNN ալգորիթմի ընդլայնված տարբերակն է, որտեղ ավելի մոտ հարևանները ստանում են ավելի մեծ կշիռ, քան հեռու գտնվողները։

Այս մոտեցման դեպքում կանխատեսման վրա ամենամեծ ազդեցությունն ունեն առավել նման դիտարկումները։

Առավելություններ.

* Ավելի ճշգրիտ է, քան սովորական KNN-ը շատ դեպքերում
* Ավելի մեծ ուշադրություն է դարձնում մոտ հարևաններին
* Կարող է բարելավել կանխատեսման որակը

Թերություններ.

* Ինչպես սովորական KNN-ը, զգայուն է feature scaling-ի նկատմամբ
* Մեծ տվյալների դեպքում կարող է աշխատել դանդաղ

Նախագծում կիրառվել է `weights='distance'` պարամետրը, որի շնորհիվ յուրաքանչյուր հարևանի ազդեցությունը հակադարձ համեմատական է նրա հեռավորությանը։


---

### 5. Decision Tree Regressor

Decision Tree Regressor-ը կառուցում է ծառաձև կառուցվածք, որտեղ յուրաքանչյուր հանգույցում տվյալները բաժանվում են ըստ որևէ հատկանիշի արժեքի։

Առավելություններ.

* Կարող է հայտնաբերել բարդ և ոչ գծային օրինաչափություններ
* Հեշտ է մեկնաբանել և պատկերացնել

Թերություններ.

* Հաճախ առաջացնում է overfitting

---

### 6. Random Forest Regressor

Random Forest-ը բաղկացած է բազմաթիվ որոշման ծառերից և հանդիսանում է Bagging մեթոդի օրինակ։

Յուրաքանչյուր ծառ ուսուցանվում է տվյալների պատահական ենթաբազմության վրա, իսկ վերջնական կանխատեսումը ստացվում է բոլոր ծառերի կանխատեսումների միջինացման միջոցով։

Առավելություններ.

* Ավելի կայուն է, քան մեկ Decision Tree-ը
* Նվազեցնում է overfitting-ը
* Տրամադրում է Feature Importance վերլուծություն

---

### 7. Gradient Boosting Regressor

Gradient Boosting-ը Boosting մեթոդի օրինակ է, որտեղ յուրաքանչյուր նոր ծառ փորձում է ուղղել նախորդ ծառերի թույլ տված սխալները։

Առավելություններ.

* Բարձր ճշգրտություն
* Կարող է նկարագրել բարդ կախվածություններ
* Հաճախ ապահովում է լավագույն արդյունքներից մեկը ռեգրեսիոն խնդիրներում

Թերություններ.

* Ավելի դանդաղ ուսուցում
* Կարող է overfit անել պարամետրերի սխալ ընտրության դեպքում

## Եզրակացություն

Նախագծի ընթացքում իրականացվել է բնակարանների գների կանխատեսում՝ օգտագործելով տարբեր մեքենայական ուսուցման մեթոդներ։

Արդյունքները ցույց տվեցին, որ գծային մոդելները կարողանում են նկարագրել տվյալների ընդհանուր միտումները, սակայն ծառային և ensemble մեթոդները սովորաբար ապահովում են ավելի բարձր ճշգրտություն։

Feature Importance վերլուծությունը ցույց տվեց, որ տան ընդհանուր որակը, մակերեսը և ավտոտնակի բնութագրերը հանդիսանում են գնի ամենակարևոր որոշիչ գործոնները։

Լավագույն արդյունքը ստացվել է Gradient boosting regression մոդելի միջոցով, որն ապահովել է ամենացածր MAE-ը և RMSE-ը։
